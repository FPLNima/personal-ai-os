# Personal AI OS

Experiments in building a personal AI operating system across ChatGPT, Codex, MCP-connected tools, Attio, Gmail, Google Calendar, ClickUp, and real personal workflows.

## Why this exists

I’m using my own work and life as the testbed for learning how AI agents, coding assistants, and connected tools can become a practical operating system.

This repo will collect experiments, notes, prompts, scripts, and small prototypes.

## Themes

- AI-native GTM workflows
- Personal CRM and relationship intelligence
- Calendar, email, and task automation
- Agentic coding with Codex, Cursor, and Claude Code
- MCP-connected workflows
- Fitness, learning, Japanese reactivation, and personal admin tools

## Status

Early public learning repo. Some work is private, some will become public as it becomes useful.
